# AlarmForce: General Idea

This is a project to intend integrate IOT Solutions in Salesforce. The stack selected:
- IOT Proxy Platform:
        - Thingspeak
-  Hardware:
         - ESP32
         - PIR Sensor
- Software:         
         - Arduino IDE
         - Salesforce:
              - Scheduled triggered flow
              - Named credential
              - External services to Thingspeak
